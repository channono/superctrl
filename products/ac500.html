Technical Specifications

AC500-V3

  • Processor: Dual-core ARM Cortex-A9, up to 800 MHz
  • Memory: Up to 32 MB program memory, 32 MB data memory
  • Communication: OPC UA, MQTT, FTP, HTTP(S), Modbus TCP/RTU
  • I/O Capacity: Up to 10,000 I/O points
  • Programming: IEC 61131-3 languages with Automation Builder
  • Security: TPM 2.0, secure boot, encrypted firmware

AC500-eco V3

  • Processor: ARM Cortex-A9, up to 600 MHz
  • Memory: Up to 16 MB program memory, 16 MB data memory
  • Communication: Modbus TCP/RTU, EtherCAT, PROFINET
  • I/O Capacity: Up to 4,000 I/O points
  • Programming: IEC 61131-3 languages with Automation Builder
  • Security: Secure boot, encrypted communication
